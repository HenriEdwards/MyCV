Java Development Projects

Food Quick - A Java Food Delivery System

Purpose of this program
  • Food Quick is a food delivery company that requires a program to manage their orders and distribute them efficiently. This project aims to create a efficient food delivery system, allowing them to track orders and assign them to drivers based on load and location. The program will also generate an invoice for customers.

  • Main.java: This file serves as the entry point for the program and is responsible for calling the relevant classes.
  • Customer.java: This class handles functionality related to requesting and storing customer information.
  • Restaurant.java: This class handles functionality related to requesting and storing restaurant information provided by the customer.

  • Meal.java: This class handles functionality related to requesting and storing meal information provided by the customer.
  • Invoice.java: This class handles the generation of the invoice based on the customer input.
  • DeliveryDriver.java: This class handles functionality related to determining the appropriate driver for the delivery based on the customer's location.
